Engram v1.0.3 shipped from PR #13: https://github.com/bbingz/engram/pull/13

**What landed:**
- Swift-native OpenClaw and Hermes adapters under `macos/Shared/EngramCore/Adapters/Sources/`, registered in `DefaultSessionAdapters`.
- Matching TypeScript adapters for MCP/Node compatibility; macOS app behavior is driven by the Swift-native adapters.
- Polycli fan-out/probe noise downgrade to `skip`.
- Cursor schema hardening: skip empty draft composers, support nested `latestConversationSummary.summary`, and ignore JSON `null` bubble rows.
- VS Code schema hardening: skip empty chat-session shell files and count real user/assistant text in Swift.
- Settings cleanup: language picker, visible embedding controls, Web UI entry points, advanced/runtime/security/observability settings, and expanded localization.

**Validation:**
- GitHub checks passed: lint, TypeScript, fixture-check, dead-code, Swift unit, UI smoke.
- Local checks passed: `npm run lint`, `npm run build`, `npm test` (119 files / 1342 tests), Swift `AdapterParityTests`, macOS Debug build.
- Read-only live adapter smoke parse passed for locally available sources; Hermes lacked a local data directory and OpenClaw had no local session files, so both are covered by fixtures and Swift native tests.
