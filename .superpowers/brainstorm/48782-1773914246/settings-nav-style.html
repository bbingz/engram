TabView 顶部标签

macOS 原生风格（类似旧版系统偏好设置）。5 个 tab 横排在顶部，点击切换内容区。

  • 和外层 Sidebar 导航视觉重复

左侧 List + 右侧 Detail

类似 macOS Ventura+ 系统设置。左边窄列表，右边内容区。

缺点

  • 已经嵌套在主 Sidebar 里，二级 List 可能视觉嘈杂
  • 内容区宽度被压缩
