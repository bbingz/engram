SwiftUI 实现方式

用语义色自动适配，无需手写 if/else：

Color(.systemBackground)   // 白 ↔ #1c1c1e
Color(.secondarySystemBackground)   // #e8e8ed ↔ #2c2c2e
Color.primary   // 黑 ↔ 白
Color.secondary   // #86868b ↔ #8e8e93
Color.green / .orange / .blue   // 系统会自动调亮度
