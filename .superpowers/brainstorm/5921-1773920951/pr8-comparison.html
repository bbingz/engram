优化后的核心优势

1

零成本可用

Ollama 本地运行，不需要任何 API key 和网络。AgentSessions 没有 AI，Readout 需要订阅

2

与 tier 系统联动

PR4 的 skip/lite 会话不浪费 AI 生成标题。只有 normal/premium 会话触发。三家都没有这种智能过滤

3

多 Provider + 多级 Fallback

AI 不可用时自动降级: generated_title → summary → 首条消息。比任何一家都更健壮

4

与现有 summary 互补

Viking summary = 长摘要 (搜索用)，generated_title = 短标题 (列表显示用)。两者共存互补
