数据采集策略

Repo 发现 从所有 session.cwd 提取唯一的 git 根目录 (git rev-parse --show-toplevel)。不像 Readout 扫全盘，我们只关注有 AI 会话的项目
状态采集 daemon 定时 (5 min) 对已知 repo 跑 git status/branch/log。结果存 git_repos 表
Active 判定 最近 24h 内有 AI 会话或有 commit = Active。7 天内 = Idle。超过 = Dormant
Quick actions open -a "Visual Studio Code" path · open -a Terminal path · git -C path pull
