实现层级

层 说明
GitContextExtractor (Node) 从会话消息中提取 environment_context 里的 git 信息 (branch, status, diff stats)
GitProbe (Node) 根据 session.cwd 运行 git 命令获取实时状态。结果缓存 60s
git_context 表 session_id, branch, last_commit, modified_count, untracked_count, status_json
HTTP API GET /api/session/:id/git → 该会话的 Git 上下文
GET /api/projects → 项目列表 + Git 摘要
Swift UI SessionDetailView: 折叠面板
ProjectsView: 项目卡片增加 branch + status

关键决策

历史 vs 实时 会话解析提取历史快照 (该会话开始时的 git 状态)。GitProbe 提供实时状态。两者都展示
全 agent 支持 不限 Codex — 任何有 cwd 的会话都可以 git probe。Claude/Gemini 的 environment_context 也解析
默认折叠 Git 面板默认折叠只显示 branch + 摘要 (3M 1U)。用户感兴趣时展开看文件列表
安全 只运行只读 git 命令 (status, log, branch)。不会修改任何仓库状态
