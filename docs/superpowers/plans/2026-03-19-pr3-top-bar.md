# PR3: Top Bar Integration — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a persistent top bar to the main window with global search (⌘K) and Resume button (right-aligned), move Theme toggle and Settings to sidebar bottom.

**Architecture:** Modify MainWindowView to wrap the NavigationSplitView detail area in a VStack with a TopBarView at the top. The top bar is always visible regardless of which page is shown. Global search reuses existing SearchPageView logic via a popover or inline overlay.

**Tech Stack:** SwiftUI, existing SearchView/SearchPageView, existing Theme.swift

**Spec:** `docs/superpowers/specs/2026-03-19-eight-prs-learning-from-agent-sessions-design.md` (PR3 section)
